
## 🚀 Nasazení

### Backend:
- Obsahuje `Procfile`, `requirements.txt`, `.env.example`
- Deployuj složku `backend/` na Railway jako **Python Web Service**

### Frontend:
- Obsahuje `vite.config.js`, `.env.example`
- Deployuj složku `frontend/` na Railway jako **Static Site**
